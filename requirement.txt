Python Environment:
	Python 3.8 or later. (recommended)

Required Python Libraries:

   
    Eel (for GUI integration with HTML/CSS/JavaScript).
    SQLite3 (for database management).
    requests (For sending HTTP requests to Mapbox)

Web Browser:

    A modern browser (e.g., Chrome, Edge, or Firefox) for rendering the GUI.

Development Tools:

    A code editor like VSCode or PyCharm is recommended

Stable internet connection (mapbox APIs are used).
